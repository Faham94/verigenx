Platform Capabilities

End-to-End Multi-Agent Architecture

A cohesive suite of agents working concurrently. From specification extraction through compilation checking and self-repair.

P1
SpecMind — Specification Parser
Processes technical spec documents (TXT, PDF, DOCX, XML, SystemRDL) using semantic chunking and ChromaDB embeddings. Extracts signals, register definitions, FSM topologies, and constraints.
chunker.py embedder.py extractor.py
P2
ArchWeaver — Dependency Graph
Builds a detailed dependency DAG of the UVM testbench component structure. Implements Kahn's topological sort for generation order, checks cycle loops, and detects signal namespace naming conflicts.
dag_builder.py resolver.py conflict_detector.py
P3
UVMForge — Testbench Synthesizer
Generates SystemVerilog UVM classes utilizing prompt libraries and transaction models. Integrates a compilation checks-based self-repair engine with multi-attempt code backups and rollback-if-worse features.
llm_filler.py repair.py generator.py
