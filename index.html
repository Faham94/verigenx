Platform Capabilities

Three Phases. Production-Ready.

Built with a modular multi-agent architecture. Each agent handles a specific stage of the verification pipeline independently.

P1
SpecMind — Specification Intelligence
Parses RTL specifications in TXT, PDF, and DOCX formats. Performs semantic chunking, ChromaDB embedding, and LLM-based extraction of signals, FSM states, registers, and timing constraints.
ingestion.py chunker.py embedder.py extractor.py
P2
ArchWeaver — Dependency Graph Engine
Constructs a directed acyclic graph of all 12 UVM components. Applies Kahn's topological sort algorithm, detects circular dependencies, identifies naming conflicts, and exports to DOT format for visualization.
dag_builder.py resolver.py conflict_detector.py
LLM
Ollama LLM Integration
Fully integrated with local Ollama models (DeepSeek Coder v2). All inference runs locally — no external API calls, no data leaves the machine. Includes response validation and centralized prompt library.
ollama_client.py prompt_library.py response_validator.py
SB
State Bus and Orchestrator
A centralized pipeline state manager with rollback support coordinates all agents. The orchestrator runs the complete Phase 1 to Phase 2 pipeline from a single command with full error handling.
state_bus.py orchestrator.py
